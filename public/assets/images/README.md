# Image placeholders

This microsite is scaffolded with placeholders so final creative assets can be dropped in without changing markup.

## Folder structure

- `logos/`: brand, footer and partner logos.
- `hero/`: key visual for the hero section.
- `badges/`: guarantee/stat badges row icons.
- `videos/`: video thumbnails and poster frames.
- `testimonials/`: customer avatars or decorative assets.

Replace placeholder files while keeping the same names to avoid template changes.
